# Простой Bash-скрипт для бэкапов

Рабочая лошадка для бэкапа сайтов на VPS. Делает дамп базы MySQL и архивирует файлы сайта. 
Ничего лишнего, только хардкор. Зато работает как часы.

## Что делает?

-   Дампит базу.
-   Архивирует папку с файлами.
-   Складывает все в папку с сегодняшней датой.
-   Чистит бэкапы старше недели, чтобы не забивать место.

## ВАЖНО!

Скрипт хранит пароль от БД в открытом виде. Убедитесь, что права на файл `backup.sh` выставлены как `700`, чтобы никто кроме вас не мог его прочитать.

## Что можно улучшить?

-   Добавить выгрузку бэкапов в облако (S3, Dropbox).
-   Прикрутить уведомления в Telegram о статусе бэкапа.
-   Сделать нормальный конфиг-файл, а не вот это всё.
